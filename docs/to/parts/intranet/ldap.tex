\subsection{LDAP}\label{ldap}

Login wordt via LDAP of Active Directory (protocol compatible) geregeld. Hiervoor wordt de \usemodule{ldap} module ingezet. Deze module voorziet in het importeren van gegevens uit de LDAP database in het Drupal profiel. Het gaat hier om de naam en functie. Aangezien LDAP op verschillende manieren ingezet kan worden zullen de precieze velden met de gemeente afgestemd moeten worden bij de gemeente specifieke implementatie.
