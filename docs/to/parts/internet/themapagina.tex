\subsection{Themapagina}\label{themapagina}

Themapagina's bevatten een overzicht van verschillende (sub)thema's. In de wireframes zijn deze opgenomen als "Subhomepagina". Een voorbeeld hiervan is:
\begin{verbatim}
http://www.woerden.nl/onderwerpen/cultuur-sport-en-vrije-tijd
\end{verbatim}
Voor deze pagina's wordt gebruik gemaakt van de \usemodule{emptypage} module. Deze module voorziet in lege pagina's. Via \usemodule{felix}\seeone{felix} wordt mogelijkheid geboden om over 3 kolommen blokken te plaatsen. De thema's zelf worden dus als redactionele blokken neergezet. Daarbij zijn de afbeelding, titel, tekst en link vrij aan te passen.
