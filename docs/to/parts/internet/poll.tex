\subsection{Poll}

Voor de poll wordt de \texttt{poll} module uit Drupal core ingezet.
Elke poll heeft een aparte pagina (detailpagina), maar het is ook mogelijk om de poll in andere pagina's op te nemen middels de \usemodule{felix} module\seeone{felix} als redactioneel blok.

De poll heeft enkel een multiple choice optie, met \'{e}\'{e}n vraag per poll.
