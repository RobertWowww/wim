\subsubsection{XML sitemap}
Via de \usemodule{Xmlsitemap} module zal automatisch een XML sitemap gegenereerd worden. Deze documenten worden door zoekmachines uitgelezen om een volledig overzicht te krijgen van alle pagina's binnen een website.

Als uitbreiding wordt de \usemodule{Dominion Xmlsitemap} module i.c.m. de \usemodule{Domain xmlsitemap} module gebruikt om sitemaps per subsite aan te bieden.
